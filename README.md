# vue-video-player
video player made by vue component

## Download
### NPM
```shell
npm i @kkan0615/vue-video-player
```
### YARN
```shell
yarn add @kkan0615/vue-video-player
```

## Props
| Name    | Type    | Default | Description                  |
|---------|---------|---------|------------------------------|
| src     | string  |         | Video Src                    |
| disable | boolean | false   | Disable to press any buttons |
|         |         |         |                              |
